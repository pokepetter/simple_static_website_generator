Simple Static Website Generator


pip install git+https://github.com/pokepetter/sswg python -m sswg


A Markdown inspired markup language.
Converts the text files in a folder into web pages.
Use the tags below to style it.


Tags

- # size 1.25 - # width 900 - # left, right, center - # normal, bold, bolder, lighter - # image imagename - # arial, times, helvetica, courier, verdana, tahoma, bookman, monospace - # background colorname - # code (start code block) - # text (end code block) - # insert some_file.txt - ## (create header with size 2) - ### (create header with size 3) - [Button Name, https://pokepetter.github.io/sswg/] - [Button Name, https://pokepetter.github.io/sswg/, image_name.jpg] (big button) - [Button Name, header name]
